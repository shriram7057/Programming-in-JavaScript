<h2>🚀 Why This Repository?</h2>

<p>
This repository helps developers learn and practice JavaScript through
real examples, structured programs, and small projects.
</p>

<ul>
<li><b>Core JavaScript</b> – Variables, loops, functions, objects, and events</li>
<li><b>Modern ES6+</b> – Arrow functions, classes, modules, async/await</li>
<li><b>DOM Manipulation</b> – Building interactive web pages</li>
<li><b>Mini Projects</b> – Calculator, To-Do list, and more</li>
<li><b>Clean Folder Structure</b> – Easy to explore and extend</li>
</ul>
